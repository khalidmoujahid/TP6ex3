Manipulation du Document Object Model (DOM)

1. Éléments de la page

Les objets globaux window et document sont les points d'entrée pour manipuler le DOM.

// Accéder aux propriétés du document console.log("Titre:", document.title); console.log("URL:", document.URL); console.log("Encodage:", document.characterSet);

2. Sélecteurs DOM

Différentes méthodes pour sélectionner des éléments dans le document.

// Sélection par ID let elem = document.getElementById('monId'); // Sélection par classe let classes = document.getElementsByClassName('maClasse'); // Sélection par balise let tags = document.getElementsByTagName('div'); // Sélecteurs CSS modernes let query = document.querySelector('.maClasse'); let queryAll = document.querySelectorAll('div.maClasse');

3. Modification du contenu

innerHTML et textContent pour modifier le contenu des éléments.

let elem = document.getElementById('monElement'); // innerHTML - interprète le HTML elem.innerHTML = '<strong>Nouveau contenu</strong>'; // textContent - texte brut seulement elem.textContent = 'Juste du texte sans balises';
Contenu initial

4. Gestion des attributs

Manipulation des attributs HTML avec getAttribute et setAttribute.

let link = document.getElementById('monLien'); // Obtenir un attribut let href = link.getAttribute('href'); // Définir un attribut link.setAttribute('target', '_blank'); link.setAttribute('data-custom', 'valeur');
Lien de démonstration

5. Modification des styles CSS

Modification directe des styles via la propriété style.

let element = document.getElementById('monElement'); // Modifier les styles element.style.color = 'red'; element.style.backgroundColor = 'yellow'; element.style.fontSize = '20px'; element.style.padding = '10px';
Élément de style démonstration

6. Gestion des classes CSS

Utilisation de classList pour manipuler les classes.

let elem = document.getElementById('monElement'); // Ajouter une classe elem.classList.add('active', 'highlight'); // Supprimer une classe elem.classList.remove('inactive'); // Basculer une classe elem.classList.toggle('visible'); // Vérifier la présence if (elem.classList.contains('active')) { console.log('Élément actif'); }
Démonstration des classes

7. Création et manipulation d'éléments

Création dynamique d'éléments et manipulation de la structure DOM.

// Créer un nouvel élément let newDiv = document.createElement('div'); newDiv.textContent = 'Nouvel élément créé'; // Ajouter au DOM document.body.appendChild(newDiv); // Supprimer un élément let oldElem = document.getElementById('ancien'); document.body.removeChild(oldElem); // Remplacer un élément let replacement = document.createElement('span'); replacement.textContent = 'Élément de remplacement'; document.body.replaceChild(replacement, oldElem);
Zone de création d'éléments

8. Gestion des événements

Utilisation de addEventListener pour gérer les interactions utilisateur.

let button = document.getElementById('monBouton'); // Ajouter un écouteur d'événement button.addEventListener('click', function() { alert('Bouton cliqué!'); }); // Événement avec paramètres button.addEventListener('mouseover', function(event) { console.log('Souris sur le bouton', event); });

9. Gestion du temps

Utilisation de setTimeout et setInterval pour l'exécution différée.

// Exécuter après délaiSSS setTimeout(function() { console.log('Exécuté après 2 secondes'); }, 2000); // Exécuter à intervalle régulier let counter = 0; let interval = setInterval(function() { counter++; console.log('Compteur:', counter); if (counter >= 5) { clearInterval(interval); } }, 1000);
